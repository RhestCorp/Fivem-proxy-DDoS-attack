# ENG

# FiveM Server Stress Tester

> **⚠️ WARNING: This tool is for educational purposes only. Unauthorized network attacks are illegal and unethical. Use only on systems you own or have explicit permission to test.**

## 📋 Overview

A multithreaded HTTP stress testing tool designed for educational security testing of FiveM servers. This tool demonstrates how distributed denial-of-service attacks work and the importance of proper server security.

## 🔧 Features

- **Multi-threaded architecture** for concurrent request handling
- **Proxy rotation** support to distribute requests
- **User-Agent randomization** to simulate legitimate traffic
- **Real-time statistics** showing request count and rate
- **Configurable parameters** for testing flexibility
- **Graceful shutdown** handling (Ctrl+C)

## 📦 Requirements

```bash
Python 3.6+
requests library
```
